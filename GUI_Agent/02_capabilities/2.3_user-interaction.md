# 3.1.3 用户交互（User Interaction）

> 论文 Section 3.1.3（Pages 15–16）

用户交互是 GUI Agent 与人类沟通的接口能力。

---

## 交互形式

### 按媒介分类

| 交互形式 | 说明 | 普及程度 |
|---------|------|---------|
| **文本（Text）** | 最常用，大多数研究采用 | 极高 |
| **语音（Voice）** | 最符合人类习惯（Siri 等商业 Agent 的主要形式） | 高（商业）/ 低（研究） |
| **GUI 交互** | 点击按钮/选择菜单与 Agent 通信 | 低 |
| **VR 交互** | 沉浸式 3D 环境 + 手势控制器 | 极低（探索阶段） |

### 按对话轮次分类

**单轮交互（Single-Turn Interaction）**：
```
用户："帮我预订佛光山附近的酒店，6天后入住，住3天，2间房，4个大人"
              ↓
Agent 自动完成任务，返回最终状态
```
要求用户**一次提供所有信息**，且需了解 GUI 环境操作方式。

**任务导向对话（Task-Oriented Dialogue）**：
```
用户："帮我预订佛光山附近的酒店"
Agent："您什么时候入住？"
用户："6天后，住3天"
Agent："需要几间房？几位成人？"
...
```
Agent 逐步收集信息，**降低用户认知负担**，提升成功率。

---

## 关键研究

| 系统 | 交互创新 |
|------|---------|
| **WebLINX** | 引入任务导向对话（Task-Oriented Dialogue）概念 |
| **Meta-GUI** | 多轮对话数据集 |
| **RUSS** | 多轮对话 + 网页任务 |

---

## Research Gap 观察

- 当前研究以**文本单轮交互**为主，多轮对话研究有限
- 用户的**长期偏好记忆**（跨会话）几乎没有被探索
- 语音交互在 LLM-based GUI Agent 中的整合研究极少
