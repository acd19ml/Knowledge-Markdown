## 学习方法

```
每个知识点的标准流程：
1. 读讲义/课件          ~20 min   建立直觉，不求全懂
2. 手推核心公式          ~30 min   哪里推不动就记下来
3. 代码复现             ~2 h      先自己写，跑不通再看参考
4. 用自己的话写笔记      ~20 min   能向后端工程师解释清楚才算懂
```

**Tutorial 使用原则（针对 CS5489 的 8 个 ipynb）：**  
关闭所有输出 → 自己逐格写代码 → 跑完再对比原版 → 记录差异和卡住点
