# 静态知识演化

> 论文第 IV-A 节

---

## 核心概念

静态知识演化将环境视为**知识库** —— 一种补充智能体内部知识的信息来源。环境不会根据智能体的行动而改变；智能体通过改进**如何**检索和使用这些静态知识来演化。

> "环境作为数据库，弥补信息空白，支持学习**事物是什么**。"

两大主要范式：
1. **智能体 RAG** —— 选择性、自适应的RAG
2. **深度研究** —— 多步骤网页搜索与综合

---

## A. 智能体 RAG

### 基础 RAG → 智能体 RAG

**传统 RAG**：查询 → 检索 → 生成（一次性）

**智能体 RAG**：智能体主动决定**何时**、**什么**以及**如何**检索 —— 检索是一种可学习的、自适应的行动。

### 关键系统

| 系统 | 关键创新 | 检索策略 |
|---|---|---|
| **FLARE**（Jiang et al., 2023） | 主动检索：仅在不确定时检索 | 不确定性触发 |
| **Self-RAG**（Asai et al., 2024） | LLM 生成特殊 token 决定何时/如何检索和自我批评 | 自我反思 token |
| **Think-on-Graph**（Sun et al., 2023） | 用 LLM 在knowledge graph上进行推理 | 图遍历 |
| **Search-R1**（Jin et al., 2025） | 通过 RL 训练 LLM 推理并利用搜索引擎 | RL 优化搜索 |
| **Search-o1**（Li et al., 2025） | 智能体搜索增强的大型推理模型 | 推理 + 搜索交叉 |
| **DSPy**（Khattab et al., 2023） | 将声明式 LM 调用编译为自我改进的流水线 | 程序化检索 |
| **R1-Searcher**（Song et al., 2025） | 通过 RL 激励 LLM 的搜索能力 | RL 训练搜索 |

### Self-RAG：带自我批评的自适应检索

```
查询 → [检索？] → 检索到的文档 → [相关？] → [批评答案？] → 输出
        ↑ token      ↑ token         ↑ token      ↑ token
     （LLM 本身生成的特殊 token）
```

Self-RAG 训练模型生成控制以下内容的特殊 token：
- 是否检索
- 检索段落的相关性
- 生成响应的有依据程度

## B. 深度研究

深度研究将智能体检索扩展到**多步骤、长时域的网页研究** —— 跨多个文档和搜索综合信息。

### 核心模式
```
初始查询
    ↓
分解为子查询
    ↓
迭代搜索 + 检索
    ↓
综合中间发现
    ↓
生成综合报告
```

### 关键系统

| 系统 | 关键创新 | 环境 |
|---|---|---|
| **MindSearch**（Chen et al., 2024） | 用多智能体深度搜索模仿人类思维 | 网页搜索 |
| **WebThinker**（Li et al., 2025） | 赋予大型推理模型深度研究能力 | 网页 + 推理 |
| **DeepResearcher**（Zheng et al., 2025） | 通过 RL 在现实世界环境中扩展深度研究 | 真实网页 |
| **WebVoyager**（He et al., 2024） | 使用大型多模态模型的端到端网页智能体 | 真实网站 |
| **MARS**（Chen et al., 2025） | 通过 MARL 协同演化的双系统深度研究 | 网页 + 多智能体 |
| **Open Deep Search** | 用开源推理智能体民主化搜索 | 网页 |

### 深度研究的质量维度
- **广度**：相关来源的覆盖范围
- **深度**：跨文档的多跳推理
- **可靠性**：引用准确性和溯源
- **综合**：多样化信息的连贯整合

---

## 智能体 RAG vs 深度研究

| 维度 | 智能体 RAG | 深度研究 |
|---|---|---|
| 范围 | 单个问题 | 复杂研究任务 |
| 检索轮次 | 1–3 次自适应 | 多次（10+） |
| 输出 | 有依据的答案 | 综合报告 |
| 关键挑战 | 何时检索 | 如何规划研究 |
| 训练信号 | 检索质量 | 研究质量指标 |

---

## 关键挑战：幻觉 vs 过度检索

```
检索不足 → 幻觉（捏造事实）
过度检索 → 上下文溢出 + 推理质量下降
```

**Self-RAG 的解决方案**：将检索训练为离散决策 → 仅在预测有助于时才检索。

**深度研究的解决方案**：层次化规划 → 每个子任务只检索所需内容。

---

## 与记忆的关联

静态知识演化与**语义记忆**密切相关（见 [Agent_Memory/02_taxonomy/2.2_cognitive-mechanisms.md](../../Agent_Memory/02_taxonomy/2.2_cognitive-mechanisms.md)）：
- RAG 将语义记忆外化到向量存储
- 深度研究使用类似工作记忆的主动工作台
- Self-RAG 增加了关于何时需要检索的元认知
